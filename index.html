Splash Wine Journal

Full stack application for Prime Digital Academy solo project

Splash Wine Journal is an alternative to pen and paper note taking for beginner wine drinkers. Users are able to add, edit, and delete wines in their own Wine List. This app was built using React, Redux, PostgreSQL, and Bootstrap.

Saga Movie Library

Weekend sprint for Prime Digital Academy

Users are able to view and add movies to their movie library. The homepage offers a basic view, while the details page shows the full details for the movie. This project was built as a practice using sagas and a redux store as well as a many to many PostgreSQL database relationship.

  • Date: November 2020

Student Feedback Portal

Weekend long sprint for Prime Digital Academy

Students are able to leave feedback for their day in class in this portal. This project was created as a way to practice using routes and redux.

  • Date: October 2020
